第二节：配置公网IP地址

May 12, 2017 • 河北工业大学

由于腾讯云采用了弹性公网IP地址，所以腾讯云为我们分配的IP地址，并没有自动的设置在服务器上。为此，我们需要进行手动的设置。

双击右下解这个小图标，打开网络和共享中心

注意，上面的演示中最后添加的IP地址：123.206.44.131，应该替换为你自己服务器的真实公网IP地址。
